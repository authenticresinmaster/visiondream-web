# 비전드림 홈페이지 — 남은 배포·연동 체크리스트

> Phase 1(랜딩·서브페이지)·Phase 2(블로그·B2B) 코드는 완료. 아래는 **회원님 액션이 필요한 인프라 연동**입니다.
> 현재 라이브: https://visiondream-web.vercel.app/ (GitHub `authenticresinmaster/visiondream-web` ↔ Vercel 자동배포)

## ☐ 1. visiondream.kr 도메인 연결
- [ ] Vercel 대시보드 → `visiondream-web` 프로젝트 → **Settings → Domains** → `visiondream.kr` 입력
- [ ] Vercel이 표시하는 DNS 레코드를 도메인 등록업체에 입력
  - 보통 루트(`visiondream.kr`)는 **A 레코드 → 76.76.21.21**
  - `www`는 **CNAME → cname.vercel-dns.com**
- [ ] (앱 서브도메인) `app.visiondream.kr` → Railway 서버로 CNAME 연결 (홈페이지의 "앱 시작하기" 링크 대상)
- ※ 등록업체(가비아/후이즈/Cloudflare 등)별 화면이 다르니, 업체 확정 시 정확한 입력값 재안내

## ☐ 2. ConvertKit(Kit) — 전자책 구독 폼 연동
- [ ] ConvertKit 계정 생성/로그인 → Form 만들기 → **Form ID** 확보
- [ ] Account Settings → Advanced → **API Key(v3)** 확보
- [ ] Vercel → Settings → Environment Variables 에 추가:
  - `CONVERTKIT_API_KEY = <발급키>`
  - `CONVERTKIT_FORM_ID = <폼ID>`
- [ ] 재배포(자동) 후 전자책 폼 제출 테스트
- ※ 키 없으면 폼은 "안전 비활성"(제출 시 에러 없이 저장만 안 됨). 대안(자체 Supabase 저장, Resend 알림)도 가능
