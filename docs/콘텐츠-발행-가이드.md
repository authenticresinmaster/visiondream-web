## 적용된 조치 (2026-07-23)

1. **push 안전망** — `git-push-safe.ps1`(계정 고정·성공 검증). 7/23에 push가 실패했는데도
   "완료"로 로깅돼 글이 로컬에만 쌓였던 사고 재발 방지. 실패 시 텔레그램 경보.
2. **사이트맵 다이어트** — `app/sitemap.ts`의 `SITEMAP_VIDEO_LANGS = ["ko"]`.
   영상 상세 en/ja 96개를 사이트맵에서 제외(deindex 아님). 색인률 60% 회복 시 되돌린다.
3. **hreflang 정확화** — `langsForPost()`로 실제 존재하는 언어만 alternates에 넣는다.
4. **내부링크 부스트** — `lib/index-status.json`(주간 자동 갱신) + `lib/index-boost.ts`.
   홈 카드(`getHomePosts`)와 관련글(`getRelatedPosts`)이 **아직 색인 안 된 글을 앞으로** 올린다.
   `needsLinks`가 비면 원래 동작(최신순·관련도순)으로 자동 복귀한다.
5. **유튜브 설명란에 글 링크** — `daily_publish_videos.py`가 업로드 후 설명란에 해당 글 URL을 붙인다.
   (nofollow라 랭킹 신호는 아니지만 발견 경로·유입을 만든다)
6. **ja 신규 발행 중단** — 색인률 25%·28일 노출 11회. 신규는 ko/en만. **기존 ja 글은 유지**(삭제·noindex 금지).

## 유튜브 Shorts — 제작 규칙

실측: 조회의 **96%가 Shorts 피드**(검색 2%). 피드 노출은 **시청지속률**이 결정한다.

| 항목 | 규칙 |
|---|---|
| 발행 리듬 | 주 3~5편 |
| 길이 | 20~35초 (한110~170자 / 영45~75단어) |
| 언어 | ko·en (ja 신규 중단과 동일 기조) |
| 승자 패턴 | laziness(지속 44%) · focus(42%) 훅·구성 재현 |

**자막은 손댈 필요 없다** — 자동자막(ASR)이 정상 생성되고 언어 인식도 정확하다.
(Data API의 `caption` 플래그는 *업로드* 자막만 세므로 false로 보이는 것뿐.)
